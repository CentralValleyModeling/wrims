Opening and Saving DTS Tree Files

To open or save a DTS tree, click on the Output tab if it's not already selected. To open a tree file click File | DTS Tree | Open and navigate to the project file to be opened. To save a tree click File | DTS Tree | Save.

Note: After opening an existing project you do not need to open a DTS tree less you want to alter the existing project or create a new project which has some or all of the tree files in common with the opened project. The DTS tree name and path associated with an open project are present in the text box next to the 'DTS Tree:' label.
