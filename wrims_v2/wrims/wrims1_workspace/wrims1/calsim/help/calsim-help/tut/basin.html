Working with Basin

The basin menus are very similar to the system menus, and work in the same way. Select the SAC/SJR/Delta toggle button first, then select a menu item and click on it, this system will perform math operations using the dss files you opened at the File menu and the pre-defined DTS associated with this menu item, and the result will be displayed in the way you selected in the View menu.

 

Example Click on San_Joaquin | Lower SJR | Vernalis Total Flow, a calculated result of its pre-defined DTS will be displayed.

Note: You should always set the project to ready first. Please see "Prepare a project for study".
