Define Statement

The define statement is similar to variable assignments and declarations in other languages. All variable identifiers must first be declared in a define statement, and thereafter used in subsequent definitions or goals. It is rather overextended in the current version, so several declaration/assignment variations exist:

  • Continuous Decision Variable
  • Binary Integer Decision Variable
  • Time Series State Variable
  • External Function
  • Constant State Variable
  • State Variable Look-up Tables
  • Sum State Variable
  • Conditional State Variable
  • Alias State Variable
